List comprehension on a file¶

codesnippets.feature27¶

Opens, reads the file line by line and closes it too (garbage collected because the stream is not assigned):

>>> lines = [a_line.rstrip() for a_line in open('examples.py')]

First lines read from the file examples.py:

>>> '\n'.join(lines[:10])
