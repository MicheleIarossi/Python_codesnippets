Boolean and/or operators¶

codesnippets.feature21¶

Boolean operators and and or return true or false objects not values!

or returns the left operand if true:

>>> 2 or 3, 3 or 2
(2, 3)

and returns the right operand if true:

>>> 2 and 3, 3 and 2
(3, 2)
