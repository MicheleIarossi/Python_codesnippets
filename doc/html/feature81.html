Static methods via function decorator @staticmethod¶

codesnippets.feature81¶

By decorating a method with @staticmethod, it can be called either via the class or via the instance and without the required self parameter.

The following class counts the instances of Cars:

class Car():
    """a car class"""
    n_val = 0
    def __init__(self,name):
        self.name = name
        Car.n_val += 1
        print('<Car: %s created>' % (name))
    @staticmethod
    def num_of_instances():
        """returns the total number of instances"""
        return Car.n_val
    def __repr__(self):
        return 'Car(%r)' % (self.name)

>>> bmw = Car('BMW 3er')
<Car: BMW 3er created>
>>> bmw
Car('BMW 3er')

>>> audi = Car('Audi A6')
<Car: Audi A6 created>
>>> audi
Car('Audi A6')

>>> vwag = Car('VW Golf')
<Car: VW Golf created>
>>> vwag
Car('VW Golf')

>>> Car.num_of_instances()
3
