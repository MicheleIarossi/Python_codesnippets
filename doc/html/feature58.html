Module imports with importlib¶

codesnippets.feature58¶

The library importlib lets us import modules via its function import_module:

>>> import importlib

>>> modname = 'module1'

>>> module1 = importlib.import_module(modname)

>>> module1.A_LIST
[4, 5, 6]
