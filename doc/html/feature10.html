Dictionaries¶

codesnippets.feature10¶

Construction with keywords:

>>> a_dic = dict(name='Bob', age=40)
{'name': 'Bob', 'age': 40}

Same as:

>>> a_dic = {'name':'Bob','age':40}
{'name': 'Bob', 'age': 40}

Keys, values and items:

>>> a_dic = {'eggs':3 ,'spam':2, 'ham':1}
{'eggs': 3, 'spam': 2, 'ham': 1}

>>> list(a_dic.keys())
['eggs', 'spam', 'ham']

>>> list(a_dic.values())
[3, 2, 1]

>>> list(a_dic.items())
[('eggs', 3), ('spam', 2), ('ham', 1)]
