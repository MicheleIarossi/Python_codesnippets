Merging dictionaries by dictionary unpacking¶

codesnippets.feature11¶

This works because **a_dict_1 and **a_dict_2 perform dictionary unpacking into keyword arguments.

>>> a_dict_1 = {'a':1, 'b':2, 'c':3}
>>> a_dict_2 = {'c':4, 'd':5, 'e':6}

>>> {**a_dict_1,**a_dict_2}
{'a': 1, 'b': 2, 'c': 4, 'd': 5, 'e': 6}

Equivalent to:

>>> dict(a=1,b=2,c=4,d=5,e=6)
{'a': 1, 'b': 2, 'c': 4, 'd': 5, 'e': 6}
