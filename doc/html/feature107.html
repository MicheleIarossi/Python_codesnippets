Type¶

codesnippets.feature107¶

Instances are made from classes, but classes are instances of type. A class object is an instance of a type object:

>>> class MyClass:
        """my class"""
        pass

>>> an_obj = MyClass()

>>> (type(an_obj), an_obj.__class__)
(<class 'codesnippets.feature107.feature107.<locals>.MyClass'>,
 <class 'codesnippets.feature107.feature107.<locals>.MyClass'>)

>>> (type(MyClass), MyClass.__class__)
(<class 'type'>, <class 'type'>)
